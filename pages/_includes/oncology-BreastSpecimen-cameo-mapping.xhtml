BC Mapping Source

This structure represents the following BC mapping definition:

oncology.BreastSpecimen maps to Specimen:
  Type maps to type
  SourceSpecimen maps to parent
  ReceivedTime maps to receivedTime
  CollectionTime maps to collection.collected[x]
  CollectionMethod maps to collection.method
  CollectionSite maps to http://hl7.org/fhir/StructureDefinition/body-site-instance
  ColdIschemiaTime maps to processing (slice on = coding.code)
