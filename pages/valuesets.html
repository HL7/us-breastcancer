ValueSets defined as part of this Implementation Guide:

Value Sets

Name Definition
AnatomicalLocationVS Codes that describe normal and pathologic anatomic systems, regions, cavities, and spaces.
BreastCancerMStageVS Metastasis category according to AJCC breast cancer staging recommendations.
BreastCancerNStageVS Regional nodes N category according to AJCC breast cancer staging recommendations. The codes do not include the stage timing prefix, recorded separately in the StageTimingPrefix attribute.
BreastCancerStageVS The AJCC stage group.
BreastCancerTStageVS Tumor category according to AJCC breast cancer staging recommendations.
BreastCancerTypeVS Histologic types of breast cancer. Codes are drawn from ICD-O-3.
BreastSiteVS Topography of the breast. Codes are drawn from ICD-O-3.
BreastSpecimenCollectionMethodVS The surgical method used to obtain the tissue sample.
BreastSpecimenTypeVS The type of specimen obtained from the breast.
ClockDirectionVS
ConditionCategoryVS
ConditionVS
DegreeOfLymphaticInvolvementVS Value set containing the degree of lymphatic involvement.
DeltaFlagVS
DiagnosticServiceVS The type of laboratory service used in producing a laboratory result
DirectionalityVS Anatomical location or specimen further detailing directionality. Codes are from NCI
ExceptionValueVS Reasons that a value associated with a test or other finding is missing. Extended set based on http://hl7.org/fhir/ValueSet/observation-valueabsentreason
HER2ISHMethodVS Value set containing In Situ Hybridization methods of determining HER2 status
HER2MethodVS Value set containing methods of determining HER2 status
HER2StatusVS HER2 status overall result
LateralityVS Anatomical location or specimen further detailing the side(s) of interest. Codes are from NCI
LeftRightLateralityVS Simplified set of laterality codes indicating detailing the side(s) of interest. Codes are from NCI
LymphSystemSubdivisionVS A lymph node area or nodal group; a subdivision of the entire lymphatic system. For breast cancer, this includes 5 areas: internal mammary lymph nodes, axillary lymph nodes level I, II, and III, and the supraclavicular lymph nodes.
MammaprintRiskScoreInterpretationVS Intepretation of Mammaprint Recurrence Score
MorphologyBehaviorVS The morphologic behavior of the cancer. These are the suffix to the ICD-O-3 histologic type codes
NottinghamCombinedGradeVS
NuclearGradeVS
NuclearPleomorphismScoreVS Value set containing the scores for nuclear pleomorphism.
OncotypeDxDCISRiskScoreInterpretationVS Intepretation of OncotypeDX DCIS (Ductal Carcinoma In Situ) Recurrence Score
OncotypeDxInvasiveRiskScoreInterpretationVS Intepretation of OncotypeDX Recurrence Score for invasive cancers.
PortionTotalityVS Further detail of the anatomical location or specimen detailing whether it includes the entire entity or only a part or portion.
PositiveNegativeVS Value set containing the values positive and negative.
PresenceContextVS A code describing whether the finding in question is present, possible, suspected, etc.
RecurrenceRiskScoreInterpretationVS Interpretations of risk as high, intermediate, or low. Answer set taken from LOINC LL3198-0
StageTimingPrefixVS When staging was done, relative to treatment events.
StagingMethodVS The methodology or standard used to do the staging.
StainingIntensityVS Part of the Allred scoring, based on the intensity of that staining, on a scale of 0 (Negative) to 3 (Strong).
TubuleFormationScoreVS Value set containing the extent (scope) of tubule formation.
TumorMarginDescriptionVS Value set containing the descriptions of tumor margin.
YesNoUnknownVS
YesNoVS
