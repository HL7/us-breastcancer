Extensions defined as part of this Implementation Guide

Name Definition
ClockDirection A direction indicated by an angle relative to 12 o'clock.
CollectionSite The body site where specimen was collected
DiagnosticService A code that classifies the clinical discipline, department or diagnostic service that created the report (e.g. cardiology, biochemistry, hematology, MRI). This is used for searching, sorting and display purposes.
DistanceToNipple The body site location in relation to the nipple.
Encounter An abstract class that represents an encounter class. This class has been stubbed out to limit the scope of this implementation guide to classes relevant to this ballot concerning breast cancer.
MorphologyBehavior A description of the morphology and behavioral characteristics of the cancer.
StageTimingPrefix Indicates when the staging was done, in terms of treatment landmarks, either clinical (at diagnosis prior to treatment) (c), pathologic (post-surgical) (p), post-neoadjuvant clinical (yc), or post-neoadjuvant pathologic(yp). The stage timing code is prefix on the N stage. For example, for post-neoadjuvant clinical staging, the N categories are preceded by 'yc' (e.g. ycNX)
base64Binary The base64Binary that represents the value of the BC element to which it is applied.
boolean The boolean that represents the value of the BC element to which it is applied.
code The code that represents the value of the BC element to which it is applied.
date The date that represents the value of the BC element to which it is applied.
dateTime The dateTime that represents the value of the BC element to which it is applied.
decimal The decimal that represents the value of the BC element to which it is applied.
id The id that represents the value of the BC element to which it is applied.
instant The instant that represents the value of the BC element to which it is applied.
integer The integer that represents the value of the BC element to which it is applied.
markdown The markdown that represents the value of the BC element to which it is applied.
oid The oid that represents the value of the BC element to which it is applied.
positiveInt The positiveInt that represents the value of the BC element to which it is applied.
string The string that represents the value of the BC element to which it is applied.
time The time that represents the value of the BC element to which it is applied.
unsignedInt The unsignedInt that represents the value of the BC element to which it is applied.
uri The uri that represents the value of the BC element to which it is applied.
xhtml The xhtml that represents the value of the BC element to which it is applied.
